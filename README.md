# cljsstarter

Starter code for a clojurescript project

## Usage

```'bash
git clone git@github.com:yayitswei/cljsstarter.git
rm -r .git
git init
lein deps
lein cljsbuild auto dev
lein run
lein vimclojure (etc..)
```
